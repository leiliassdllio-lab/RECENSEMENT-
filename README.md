# Herisson

Projet web statique autour du recensement des herissons.

## Structure

Pages HTML a la racine :
- `index.html` : page d'accueil
- `login.html` : connexion / creation de compte
- `recensement.html` : formulaire principal de recensement
- `quiz.html` : quiz
- `cartographie.html` : carte des observations
- `conseils.html` : conseils pratiques
- `profil.html` : profil utilisateur
- `fiche-scientifique.html` : fiche scientifique
- `confirmation.html` : confirmation apres envoi

Anciennes pages dans `legacy/` :
- `legacy/recensement-legacy.html`
- `legacy/quiz-legacy.html`
- `legacy/equipe.html`

Ressources classees dans `assets/` :
- `assets/css/` : feuille de style partagee
- `assets/images/badges/` : badges du profil
- `assets/images/cards/` : images des cartes de l'accueil
- `assets/images/observations/` : etats du herisson
- `assets/images/environment/` : types d'environnement
- `assets/images/science/` : images de la fiche scientifique
- `assets/media/` : video et fond

## Notes

- Les pages du dossier `legacy/` sont secondaires ou conservent d'anciennes versions.
- La navigation principale part de `index.html`.
